# 專案目標說明

## 任務概述
製作一個靜態 Landing Page，使用純 HTML + CSS（Flexbox）完成，不需要 JavaScript，不需要 RWD（響應式設計），只需在桌機版面正常顯示即可。

---

## 主題與品牌
這是一間名為 **Core & Calm Studio** 的社區型功能性健身教室的行銷首頁。
主打族群為 30 歲以上、重視長期健康與抗老化的學員。
教室風格親民、無壓力，器材以軟墊、啞鈴、壺鈴、彈力帶、TRX 為主，不使用槓鈴等大型器械。

## 頁面架構（共 6 個區塊）

1. **Header** — Logo 文字 + 導覽連結（深色背景）
2. **Hero** — 主標題 + 副標題 + CTA 按鈕 + 右側圖片（深色背景）
3. **特色介紹區** — Section 標題 + 4 張 Card（白色背景）
4. **Quote 區** — 學員見證引言（淺灰背景）
5. **CTA 區** — 行動呼籲標題 + 按鈕（藍色背景）
6. **Footer** — 版權文字（深色背景）

---

## 開發規則
- 字型：Roboto（透過 Google Fonts 引入）
- 排版方式：全站使用 Flexbox
- CSS：單一 `style.css` 檔案
- 圖片：放在 `images/` 資料夾，請先確認圖片已備妥再開始撰寫 HTML
- 不需要 pixel-perfect，版面比例與各元素相對位置正確即可

---

## 工作流程建議
1. 閱讀 `02-content.md` 了解所有文案
2. 閱讀 `03-design-spec.md` 確認色彩與字型規範
3. 確認 `04-images.md` 中的 5 張圖片已放入 `images/` 資料夾
4. 由上而下逐區完成 HTML 結構，再套用 CSS 樣式
