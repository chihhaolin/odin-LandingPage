# Core & Calm Studio — Landing Page

**Core & Calm Studio** 是一間社區型功能性健身教室的行銷首頁，專為重視長期健康與抗老化的族群設計。

本專案為靜態 Landing Page，使用純 HTML + CSS（Flexbox）實作，無 JavaScript，以桌機版面為主。

---

## 專案結構

```
odin_LandingPage/
├── index.html       # 主頁面
├── style.css        # 全站樣式
├── images/          # 圖片素材
│   ├── hero.jpg
│   ├── card-strength.jpg
│   ├── card-cardio.jpg
│   ├── card-antiaging.jpg
│   └── card-coaching.jpg
└── ref/             # 設計規範文件
```

---

## 頁面區塊

| 區塊 | 說明 |
|------|------|
| Header | Logo + 導覽連結 |
| Hero | 主標題、副標題、CTA 按鈕、右側大圖 |
| 特色介紹 | 4 張課程卡片 |
| Quote | 學員見證引言 |
| CTA | 行動呼籲 + 預約按鈕 |
| Footer | 版權聲明 |

---

## 技術規格

- **字型**：Roboto（Google Fonts）
- **排版**：全站 Flexbox
- **色彩**：依品牌設計規範（主色 `#1F2937`、藍色 `#3882F6`）
- **RWD**：無（桌機版面）
